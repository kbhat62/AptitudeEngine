What is Aptitude Engine?
Aptitude engine is a smart apptitude question solver engine which can be used to solve "Number Series", "Speed Distance and Time" , "Clocks" apptitude problems. It provides the result along with easy steps to solve the problem.
How to input questions?
Apptitude Engine can process english language! That means you can input questions in english language.But be carefull with your grammer.
How to input a series for missing number problems?
You need to follow standard format to find a missing number in a series. The format is - num1,num2,-,num3,num4,... where '-' is the missing number to find
